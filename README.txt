Edge of the Force: Jedi vs Sith — готовый статический сайт

Как залить на хостинг:
1. Распакуйте архив.
2. Загрузите содержимое папки eotf_jvs_site_ready на хостинг в public_html / www.
3. Важно: файл index.html должен лежать прямо в корне сайта.

Структура:
index.html
assets/style.css
assets/script.js

Что заменить:
- connect 0.0.0.0:27015 — на реальный IP сервера
- discord.gg/your-link — на реальную ссылку Discord
- Steam Workshop — на ссылку коллекции аддонов
- href="#" у документов — на ссылки Google Docs / Notion / Discord / GitBook

Сайт не требует Node.js, React, npm или сборки. Это готовая статическая версия.
